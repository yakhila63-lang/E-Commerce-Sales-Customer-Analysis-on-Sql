 E-Commerce Sales & Customer Analysis using SQL

## 📊 Project Overview
Analyzed ₹7.34L e-commerce data using MySQL to find revenue patterns and operational gaps.

## 🔍 Key Insights
1. **Total Revenue: ₹7,34,290** from 17 orders
2. **91% revenue from Hyderabad** → ₹5.82L from 3 customers only
3. **₹5.04L orders in Processing** → Fulfillment opportunity
4. **29% delivery rate** → Operational bottleneck

## 🛠️ Tech Stack
MySQL Workbench, SQL: JOINs, GROUP BY, Aggregations
